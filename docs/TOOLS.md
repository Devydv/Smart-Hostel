# Tools Used

This project uses the following tools and services for development, deployment, and monitoring.

## Application

- Flask (web framework)
- MySQL 8 (database)
- Gunicorn (WSGI server)

## DevOps and Monitoring

- Docker + Docker Compose (local stack and deployment)
- Prometheus (metrics collection)
- Grafana (dashboards)
- mysqld-exporter (MySQL metrics)
- node-exporter (host metrics)

## CI/CD

- GitHub Actions (CI/CD pipeline)
- Ansible (EC2 deployment automation)

## Quality

- Ruff (linting)
- Pytest (tests)
- Pytest-cov (coverage report)
